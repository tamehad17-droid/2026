# Environment Variables

Create a `.env` file in this folder with the following keys:

```
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=

# AdGem
VITE_ADGEM_APP_ID=

# AdSterra
VITE_ADSTERRA_PUBLISHER_ID=
VITE_ADSTERRA_PLACEMENT_ID=
VITE_ADSTERRA_DIRECT_URL=
```

Notes:
- Vite reads variables via `import.meta.env.VITE_*`.
- Do not commit real keys.
